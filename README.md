# 利用koa框架写一个登录的小案例

### 第一步

```sh

npm install

```

### 第二步

```sh

在根目录找到test_koa.sql数据库文件并导入mysql数据库

```

### 第三步

```sh

在routes目录找到mysql.js文件 配置数据库参数

```

### 第四步启动服务器

```sh

node ./www/bin   

or 

npm start

```


### 最后访问

```sh

http://localhost:3000/

```
